# SimpleReadWriter
Conf file reader and writer
<br><br>

# Dependencies
1. libgcc_s_dw2-1.dll
2. libstdc++-6.dll
<br><br>

# Usage
run the app, input the keywords to read its value inside conf file and write to record.txt
<br><br>

# Build Requirement
1. mingw
2. nodejs
2. npm
```
$ npm run build
$ ./app.exe 
```
